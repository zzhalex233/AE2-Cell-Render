# AE2 Cell Render - 1.12.2

---
## Mod
- Brings a more modern-style cell color rendering experience to the AE2 1.12.2 `ME Drive`

## Features

- Renders each occupied `ME Drive` slot using the main color of the inserted storage cell
- Hides the color overlay when the drive is not connected to an active ME network
- Calculates the main color by analyzing the cell texture, with support for modded cells and current resourcepacks
- Unifies main colors across the same cell series, including capacity-suffix variants and same-item metadata variants
- Can split one series into color families so cells with clearly different base colors do not get forced into one shared color
- Exposes config options for series or family grouping and final display-color enhancement

## Installation & Dependencies

- Install on both client and server

- Required: [AE2UEL](https://github.com/AE2-UEL/Applied-Energistics-2)
- Required: [MixinBooter](https://github.com/CleanroomMC/MixinBooter)
